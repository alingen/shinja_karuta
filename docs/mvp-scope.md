# 信者カルタ — MVPスコープ

## 実装する

- CPU との 1 対 1 競技カルタ（ローカル完結）
- 暗記画面（自陣・敵陣 10 枚表示、カウントダウン）
- 対戦画面（札選択、音声再生、正誤判定）
- 送り札選択 UI
- 空札・沼札ロジック
- お手つき / 両者お手つき処理
- 同着判定
- 決まり秒の記録
- 試合結果画面
- ブラウザ SpeechSynthesis による仮音声再生
- 動作確認用仮カードデータ（約 25 枚）
- ゲームエンジンの単体テスト
- ゲーム状態の React Context 管理

## 実装しない（将来対応）

- Supabase 接続（Auth / DB / Realtime / Storage）
- オンライン対戦
- レート計算・ランキング
- アカウント登録・プロフィール
- 広告・課金
- ユーザー投稿・カード作成
- 管理画面
- ネイティブアプリ（iOS / Android）
- 複数の CPU 難易度（Easy / Hard）
- 暗記時間中の札の並び替え
- 詳細なアニメーション・エフェクト
- BGM・SE

## 技術的な仮定

- 音声は `audioUrl` がない場合 SpeechSynthesis で代替再生
- CPU 難易度は Normal 固定
- デッキは仮データ 1 種のみ
- カードの `displayText` を SpeechSynthesis に渡す
