# 信者カルタ — プロダクト概要

## サービス概要

「信者カルタ」は、配信者の名場面や発言を題材にした 1 対 1 のデジタル競技カルタゲームです。
競技かるたのルールをベースに、スマートフォン・PC 両対応のブラウザゲームとして提供します。

## 対象ユーザー

- 特定配信者の視聴者・ファン
- 配信文化・VTuber 文化に親しむユーザー
- カードゲームや反射神経ゲームを好むユーザー

## 汎用カルタ基盤としての展開

現時点では特定の配信者向けコンテンツを使用していますが、コード・DB・ドメインモデルは
**特定配信者に依存しない**設計を維持します。

将来的には以下を目指します。

- 異なる配信者・VTuber・コンテンツ向けにデッキを差し替えられる汎用基盤
- デッキ単位でコンテンツを追加・販売できる仕組み
- マルチテナント対応（配信者ごとのブランド化）

## 売却可能な構造

- ドメインモデルが特定人物に依存しない (`deckId` でコンテンツを分離)
- ゲームエンジンが UI・インフラから独立している
- Supabase などのマネージドサービスへの移行が容易
- 収益モデル（課金・広告・スポンサーシップ）を後から追加可能な構造

## 仮定

- 初期デッキは動作確認用の仮データを使用
- 将来のコンテンツは `deckId` + `isBoardEligible` フラグで管理
